
# Welcome bot

## Clone
```
git clone https://github.com/yusufjonnazarov/WelcomeBot.git
```
## Direct to project
```
cd WelcomeBot
```
## Install Requirements
```
npm i
```
## Fill some info with creating `.env` file
```
BOT_TOKEN=bot-token
WELCOME_MESSAGE=welcome-message
LEFT_MESSAGE=left-message
GROUP_ID=group-id
```
## Run project
```
npm run dev
```
