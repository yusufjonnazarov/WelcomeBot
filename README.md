
# Welcome bot

## Clone
```
git clone https://github.com/yusufjonnazarov/WelcomeBot.git
```
## Direct to project
```
cd WelcomeBot
```
## Install Requirements
```
npm i
```
## Fill some info with creating `.env` file
```
BOT_TOKEN=bot_token
WELCOME_MESSAGE=welcome_message
LEFT_MESSAGE=left_message
GROUP_ID=group_id
```
## Run project
```
npm run dev
```
